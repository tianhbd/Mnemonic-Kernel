# Changelog

All notable changes to Mnemonic-Kernel will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.2.0] - 2026-06-25

### Added
- MIT License file
- CONTRIBUTING.md with contribution guidelines
- CODE_OF_CONDUCT.md
- SECURITY.md with security policy
- GitHub issue templates (bug report, feature request)
- Pull request template
- GitHub Actions CI workflow
- GitHub Actions release workflow
- GitHub funding configuration
- Badge section in README

### Changed
- Improved README with installation quickstart, contributing guide, and project stats badges
- Enhanced documentation structure

## [0.1.0] - 2026-04-30

### Added
- Initial release
- Core four-layer architecture (AGENTS.md, memory, skills, journal)
- OpenCode adapter layer
- PowerShell management scripts (check, deploy, journal, memory, skill)
- Verification tooling
- Template system for new skills and memories
- Bilingual documentation (Chinese/English)
