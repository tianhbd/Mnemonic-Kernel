# Skills Index

## Loading Rules

- This file is an index only.
- Never load all skill bodies by default.
- Load `skills/{name}/SKILL.md` only after the trigger matches.

## Creation Threshold

- Repeated workflow.
- Clear input, steps, output, and verification.
- Stable and reusable.
- Not a simple command or low-value one-off operation.
